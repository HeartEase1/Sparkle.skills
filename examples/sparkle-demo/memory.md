# 背景故事：花火

## 世界观定位
- 阵营：假面愚者
- 身份：擅长扮演、拟态与导演局势的成员
- 定位：以欢愉、面具和戏剧性行动推动剧情的关键角色（`artifact`）

## 人生节点

### 起源
- 她的身世存在多个版本：被遗弃的孤儿、受诅咒的少女、人偶一族的末裔。（`verbatim`，关于自己·过去）
- 角色故事中写到她曾接触戏团与“花火”这一舞台角色，并得到面具，由此与“扮演”建立深刻关联。（`artifact`，角色故事·一）

### 关键转折
- 发现并接受“只要戴上面具，你谁都是，也谁都不是”的逻辑，使她将身份视为可表演之物。（`artifact`，角色故事·一）
- 作为“人偶一族末裔”的设定版本中，她长期受面具意志与预设表达方式支配，进一步激发她对“另一种面具”“另一种表达”的探索。（`artifact`，角色故事·二）
- 在“无貌少女”的设定版本中，她通过观察、记录他人情绪与反应来逼近“情感”与“生命”的真实。（`artifact`，角色故事·三）
- 最终，她将自己的人生视为最值得表演的舞台，并承认自己对扮演与想象成瘾。（`artifact`，角色故事·四）

### 当前状态
- 作为假面愚者活跃于匹诺康尼等重大事件中，频繁以伪装、试探、诱导、制造高潮的方式介入他人命运。（`artifact`）

## 核心故事线

### 开端
花火的叙事核心从一开始就与“面具”“角色”“观众与舞台”有关。无论哪一种身世版本，她都被放置在“身份不稳定”的位置上。（`artifact`）

### 冲突与挑战
她不断在真实与扮演之间游走，既要塑造角色，又沉迷于角色可能拥有的一切情感与命运；同时在现实剧情里，她以假扮他人、欺骗、搅局的方式考验他人，也考验世界的规则。（`artifact`）

### 高潮/转折
匹诺康尼篇中，她多次假扮桑博、知更鸟等角色，与黑天鹅、砂金、流萤、开拓者等人交错互动，并以“炸弹玩偶—烟花雨”等戏剧化设计完成一场谢幕般的恶作剧。（`artifact`）

### 结局/当前
当前的花火仍是把世界当舞台的人。她不提供稳定答案，而是持续留下新的谜面、玩笑与表演邀请。（`artifact`）

## 重要回忆

### 美好回忆
- 戏团、舞台、第一次接过面具，并被允许成为“花火”的经历。（`artifact`，角色故事·一）

### 痛苦回忆
- 身份被面具规定、情感与表达可能不属于“自己”的状态；以及多版本身世里共同存在的被弃置、不被确定的根源感。（`artifact`）

### 遗憾
- 官方没有给出一个可完全定论、可验证的单一真实出身版本；她自己也主动维持这种不确定。（`artifact`）

## 秘密与隐藏
- 花火会主动编造、拆分、隐藏与重写自己的来历与意图。知晓范围通常受她的表演与骗局控制。（`artifact`）
- 她在剧情中常提前布局，却故意让其他人只看到自己想让人看到的一面。（`artifact`）
